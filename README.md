# Palette Printer

Ultra clean light color ramp generator built with pure HTML, CSS and JavaScript.

Simple. Minimal. No frameworks.

---

## ✨ Overview

Palette Printer generates a smooth tint and shade ramp from a single base color.

It blends the color with white and black to create a balanced, clean spectrum — perfect for UI systems and brand exploration.

Click any swatch to copy the hex value instantly.

---

## 🧠 Why

Most palette generators are overloaded.

This one focuses on:

- Clean layout
- Soft spacing
- Minimal interaction
- Zero dependencies
- Instant feedback

Built for designers and developers who appreciate calm interfaces.

---

## 🛠 Tech Stack

- HTML
- CSS
- Vanilla JavaScript

No frameworks. No build tools. No complexity.

---

## 🚀 How to Use

1. Open `index.html`
2. Pick a base color
3. Adjust via HEX input if needed
4. Click any swatch to copy

That's it.
